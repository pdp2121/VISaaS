flaskr-bdd
==========

Flask Behaviour Driven Design Example, using the flask tutorial at:

http://flask.pocoo.org/docs/tutorial/

and behave

https://github.com/behave/behave

## Running the server
1. Create virtualenv
2. Install requirements
```
pip3 install -r requirements.txt
```
3. Run server
```
python3 flaskr.py
```

## To Test
Run behave
```
behave
```
