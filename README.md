## To run the app
1. Create virtualenv
2. Install requirements
```
pip3 install -r requirements.txt
```
3. Run server
```
python3 app.py
```

## To Test
Run behave to test user stories
```
behave
```
Run test coverage
```
coverage run -m unittest tests
coverage report --omit=tests.py
```
